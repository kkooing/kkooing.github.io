## 0. MathJax 기본
### 0.1 MathJax 와 LaTeX의 차이
참고  
[Differences from Actual TeX](http://docs.mathjax.org/en/latest/input/tex/differences.html)  
[wikibooks LaTeX/Mathematics](https://en.wikibooks.org/wiki/LaTeX/Mathematics)  
[onemathematicalcat](https://www.onemathematicalcat.org/MathJaxDocumentation/TeXSyntax.htm)  
[onemathematicalcat(한글 번역)](https://www.onemathematicalcat.org/MathJaxDocumentation/MathJaxKorean/TeXSyntax_ko.html)


&nbsp;MathJax는 실제 Tex/LaTeX의 수학 모드만 지원한다. 두 가지 예외가 있는데 첫째는 `\ref` 이고, 둘째는 `$`기호를 이스케이프 하기 위한 `\text{}`이다.

### 0.2 MathJax 입력
&nbsp;MathJax 입력 방법으로는 display 모드를 위한 `\[...\]`, `$$...$$`와 text 모드를 위한 `\(...\)` 가 있다. text 모드의 다른 표현인 `$...$`를 사용하려면 아래처럼 설정해야한다.

```js
window.MathJax = {
  tex: {
    inlineMath: [['$', '$'], ['\\(', '\\)']]
  }
};
```

&nbsp;수식 내부에서 `\begin{equation}...\end{equation}` 같은 환경명령을 실행할 수 있다. (관련 성정: processEnvironments [TeX Input Processor Options](http://docs.mathjax.org/en/latest/options/input/tex.html#tex-options))

### 0.3 HTML에서의 주의사항
&nbsp;MathJax 구문 안에서 &lt;, &gt;, &amp; 를 입력하려면 앞뒤로 공백을 두거나 HTML entity 를 사용한다.

### 0.4 TeX 매크로
&nbsp;`\def`, `\newcommand`, `\renewcommand`, `\newenvironment`, `\renewenvironment`, `\let` 를 사용할 수 있다.

### 0.5 자동 방정식 번호 지정
&nbsp;기본적으로 꺼져있지만 아래 설정으로 활성화한다.

```js
window.MathJax = {
  tex: {
    tags: 'ams'
  }
};
```

&nbsp;`notag`또는 `\nonumber`로 개별적으로 번호를 끄거나, `\tag{}`로 수동 번호지정이 가능하다.

&nbsp;ams 환경에서는 `*`로 번호를 끄고 켤 수 있다.
번호 O

```js
\begin{equation}
   E = mc^2
\end{equation}
```

번호 X

```js
\begin{equation*}
   e^{\pi i} + 1 = 0
\end{equation*}
```

&nbsp;`\label`과 `\ref` 또는 `\eqref`로 방정식 참조하는법

```js
In equation \eqref{eq:sample}, we find the value of an
interesting integral:

\begin{equation}
  \int_0^\infty \frac{x^3}{e^x-1}\,dx = \frac{\pi^4}{15}
  \label{eq:sample}
\end{equation}
```

text 모드 (인라인): `\(`, `\)`  
문장 내부에 text 모드 \\( \LaTeX \\) 가 들어있다.  
  
displayed 모드 (블록): `\[`, `\]`<br/>
아래에는 display 모드
<div class="cf-scroll">
\[ \LaTeX \]
</div>
가 들어있다.  

<div class="note" markdown=1>
  <div class="title" markdown=span>kramdown 에서 `\(`, `\)`</div>
kramdown 에서는 `\` 를 이스케이프 문자로 사용하기 때문에 `\(`, `\)` 대신에 `\\(`, `\\)` 를 사용해야 한다. HTML 태그 안에서는 kramdown 문법이 작동하지 않기 때문에 `\` 를 한 개만 써도 된다.
</div>

## 4. 분수

<div class="cf-scroll">
  <table>
    <tr><td markdown=span>`\frac{a}{b}`</td><td>\(\frac{a}{b}\)</td></tr>
  </table>

분수는 가독성을 위해 display 모드보다 text 모드일 때의 크기가 더 작다. `\tfrac`, `\dfrac` 을 사용해 이 크기를 강제할 수 있다.

<div class="cf-scroll">
  <table>
    <tr><td markdown=span>`\tfrac{a}{b}`</td><td>\(\tfrac{a}{b}\)</td></tr>
    <tr><td markdown=span>`\dfrac{a}{b}`</td><td>\(\dfrac{a}{b}\)</td></tr>
  </table>
</div>

번분수를 표현할 때는 `\frac` 대신에 `\cfrac` 을 사용해 가독성을 높일 수 있다.

```
\frac{a}{b+\frac{c}{d+\frac{e}{f+\frac{g}{h}}}}
```

<div class="cf-scroll">
\[\frac{a}{b+\frac{c}{d+\frac{e}{f+\frac{g}{h}}}}\]
</div>

```
\cfrac{a}{b+\cfrac{c}{d+\cfrac{e}{f+\cfrac{g}{h}}}}
```

<div class="cf-scroll">
\[\cfrac{a}{b+\cfrac{c}{d+\cfrac{e}{f+\cfrac{g}{h}}}}\]
</div>
